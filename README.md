# Dodge the Creeps! 躲避敌人！

> Godot 官方 2D 教程练习

这是一个基于 Godot 官方 2D 教程 **[Dodge the Creeps!](https://docs.godotengine.org/zh-cn/4.x/getting_started/first_2d_game/index.html)** 制作的练习项目，使用 Godot C#/.NET 实现。玩家需要控制角色在屏幕内移动，尽可能长时间躲避不断从四周生成的敌人；存活时间越久，分数越高。

## 项目信息

- 引擎版本：Godot 4.6 / Godot.NET.Sdk 4.6.3
- 脚本语言：C#
- 目标框架：.NET 8.0
- 主场景：`stages/main/main.tscn`
- 默认窗口：480 x 720
- 玩法类型：2D 躲避生存小游戏

## 玩法说明

点击开始按钮后，游戏会进入准备倒计时。倒计时结束后，敌人会沿屏幕边缘随机生成，并向场景内部移动。玩家碰到任意敌人后游戏结束，界面会显示结束提示并提供重试按钮。

分数按存活时间累计，当前实现为每秒增加 1 分。

## 操作方式

- 移动：方向键 `↑` `←` `↓` `→` 或 `W` `A` `S` `D`
- 手柄移动：十字键 / 左摇杆
- 开始或重试：点击开始或重试按钮 / 键盘回车 `Enter` / 手柄 `A`

## 运行项目

1. 安装 Godot 4.6 .NET 版本。
2. 使用 Godot 打开本项目根目录。
3. 首次打开后构建 C# 项目。
4. 运行主场景或直接运行项目。

也可以使用解决方案文件 `Dodge the Creeps.sln` 在支持 .NET 的 IDE 中查看和编辑 C# 代码。

## 项目结构

```text
.
├── project.godot
├── Dodge the Creeps.csproj
├── Dodge the Creeps.sln
├── stages/
│   └── main/          # 主场景与游戏流程控制
├── entities/
│   ├── player/        # 玩家场景、移动和碰撞逻辑
│   └── mob/           # 敌人场景、随机动画和离屏销毁逻辑
├── ui/
│   └── hud/           # 分数、提示、开始和重试界面
└── assets/
    ├── audio/         # 背景音乐与游戏结束音效
    └── fonts/         # Xolonium 字体及许可文件
```

## 核心实现

- `Main.cs` 负责游戏流程：开始新游戏、重置分数、生成敌人、计时得分和处理游戏结束。
- `Player.cs` 负责玩家输入、动画切换、边界限制和碰撞信号。
- `Mob.cs` 负责敌人随机动画，并在离开屏幕后自动释放。
- `HUD.cs` 负责显示分数、提示文本、开始按钮和重试按钮。
